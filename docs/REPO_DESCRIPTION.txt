S&P 500 insider selling tracker with composite anomaly scoring and local LLM explanations. Go + Rust + React + Python (FastAPI/Ollama). Aggregates Form 4 filings from EDGAR, FMP, SEC-API, EODHD; scores anomalies via volume, breadth, and acceleration signals. Configurable trend periods (1D/1W/1M/3M) with price charts.
